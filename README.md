# Mindfolk-RPG
It is project for Yandex course from Kudinov Konstantin and Andreev Konstantin

Техническое задание проекта
Название проекта: Mindfolk RPG
Жанр: 2D RPG (top-down)

1. Общая идея игры
Игрок управляет персонажем, который путешествует между деревней и поляной. В деревне игрок взаимодействует с жителями, получает квесты на убийство мобов или сбор трав. На поляне выполняет эти квесты, сражаясь с мобами и собирая ресурсы. Особенность игры — динамические диалоги и логика NPC, формируемые через DeepSeek API.
2. Основные механики
•	Передвижение игрока с помощью клавиш WASD
•	Взаимодействие с NPC для получения квестов
•	Выполнение квестов: убийство мобов и сбор трав
•	Динамические диалоги с NPC через DeepSeek API
•	Ограниченное перемещение NPC по локации (поведение ИИ)
3. Игровые объекты
Игрок:
•	Может перемещаться по локациям
•	Может собирать травы
•	Может атаковать мобов
•	Получает и выполняет квесты
NPC (жители деревни):
•	Имеют имя и роль
•	Генерируют диалоги через ИИ
•	Могут выдавать квесты
•	Ограниченно перемещаются по деревне
Мобы:
•	Враждебные сущности на поляне
•	Имеют здоровье и наносят урон игроку
•	Уничтожение мобов засчитывается для квестов
Травы:
•	Статические объекты на поляне
•	Сбор трав засчитывается для квестов
4. Локации
•	Деревня — стартовая локация, содержит NPC, точки диалогов, выдачу квестов
•	Поляна — зона выполнения квестов, содержит мобов и травы
•	В будущем планируется добавление новых локаций
5. Интерфейс
•	Стартовое меню с кнопкой начала игры
•	Игровой HUD: отображение активного квеста и прогресса
•	Экран завершения квеста или уровня
•	Сообщения от NPC в диалоговом окне
6. Звук и анимация
•	Анимация движения игрока и NPC
•	Звуки сражений (атака, получение урона)
•	Звуки взаимодействия с NPC и сбора трав
7. Используемые технологии
•	Библиотека: Arcade
•	Интеграция DeepSeek API для генерации диалогов
•	Хранение состояния игры и квестов в JSON
•	Ограничение на количество одновременно активных NPC для производительности
