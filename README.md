# Mindfolk RPG

**Mindfolk RPG** — это 2D top-down RPG-игра, созданная в рамках курса Яндекса.
Проект разрабатывается **Кудиновым Константином** и **Андреевым Константином**.

Главная особенность игры — **динамические диалоги NPC**, генерируемые с помощью **Google API** и **OpenAI API**.

---

# Описание игры

Игрок управляет персонажем, который путешествует между:

* **Деревней** — получение квестов, общение с NPC
* **Поляной** — выполнение квестов: убийство мобов и сбор трав

Игровой процесс построен вокруг:

* исследования локаций
* взаимодействия с NPC
* выполнения заданий
* боёв с мобами
* сбора ресурсов

---

# Основные механики

* Передвижение персонажа с помощью **WASD**
* Взаимодействие с NPC и получение квестов на клавишу E
* Выполнение квестов:

  * убийство мобов
  * сбор трав
* **ИИ-диалоги NPC через Google API**
* Ограниченное поведение NPC (перемещение по деревне)

---

# Игровые объекты

## Игрок

* Перемещается между локациями
* Сражается с мобами
* Собирает травы
* Получает и выполняет квесты
* Зарабатывает монеты

## NPC

* Имеют имя и роль
* Генерируют диалоги через ИИ
* Выдают квесты
* Ограниченно перемещаются

## Мобы

* Враждебные сущности на поляне
* Имеют здоровье
* Наносят урон игроку
* Засчитываются в квестах

## Травы

* Статические объекты
* Собираются для выполнения квестов
